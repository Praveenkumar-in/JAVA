# ☕ Java Tutorial Programs — Basic to Advanced

This repository contains a structured collection of **Java programs** covering concepts from **basic to advanced level**.  
It is created for **learning, practice, and interview preparation**.

---

## 📘 Topics Covered

- Java Basics (Syntax, Variables, Data Types)
- Operators & Control Statements
- Loops & Conditional Logic
- Methods & Functions
- Arrays & Strings
- Object Oriented Programming (OOP)
- Classes & Objects
- Inheritance & Polymorphism
- Encapsulation & Abstraction
- Exception Handling
- Collections Framework
- File Handling
- Logical & Practice Programs
